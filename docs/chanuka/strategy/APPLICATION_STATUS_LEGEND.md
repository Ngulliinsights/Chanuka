### From Goethe-stitut Application

**Insight:** Positioning Chanuka as *cultural practice* (not pure tech) created differentiation opportunity

**Lesson:** Multiple entry points for same organization:
- Tech foundations → technical capability angle
- Cultural foundations → artistic/cultural angle
- Democracy foundations → civic accountability angle

**Implication:** Each application should be carefully framed to match funder's actual theory of change
